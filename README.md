# AI-Powered Skin Disease Detection and Recommendation System

## Project ID
25-26J-409

## Project Overview
This project focuses on developing a mobile-based AI system for skin disease detection and recommendation using CNN and Generative AI technologies.

The system helps users identify possible skin diseases using image analysis and provides personalized recommendations with explainable AI support.

## Main Features
- Skin disease image classification
- CNN-based disease detection
- Explainable AI recommendations
- Retrieval-Augmented Generation (RAG)
- Mobile application support
- Lesion heatmaps and visualization
- Privacy-focused architecture

## Technologies Used
- Python
- TensorFlow / PyTorch
- Flutter
- FastAPI
- OpenCV
- Firebase
- MongoDB / PostgreSQL

## Datasets
- ISIC Dataset
- HAM10000 Dataset
- Derm7pt Dataset
- PH2 Dataset

## Objectives
- Improve fairness across different skin tones
- Generate explainable predictions
- Provide personalized recommendations
- Support mobile-first healthcare access
