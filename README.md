# AI-Powered-Skin-Disease-Detection-and-Recommendation-System PP1 Presentation

## Project Overview
The Skin Disease Detection and Recommendation System is an AI-powered research project developed to improve early skin disease identification and provide educational support for users. The system focuses on accessibility, fairness, explainability, and personalized recommendations through deep learning and retrieval-based AI techniques.

This project was developed as part of the undergraduate research project for the BSc (Hons) Information Technology degree program.



##  Research Problem
Skin diseases affect millions of people worldwide, while access to dermatologists remains limited in many regions. Existing AI-based diagnosis systems mainly focus on classification and often lack:

- Explainable predictions
- Severity stage prediction
- Fairness across different skin tones
- Offline accessibility
- Personalized recommendations

This project addresses these limitations by introducing an explainable and stage-aware AI solution.



##  Main Objectives

- Detect multiple skin diseases using deep learning
- Predict severity stages of detected diseases
- Provide explainable AI outputs
- Support fair performance across different skin tones
- Deliver educational and recommendation-based responses
- Improve accessibility through a mobile-friendly approach



## Key Features

###  Skin Disease Classification
- Detects 7 categories of skin diseases
- Uses deep learning-based image classification
- Optimized for scalable deployment

###  Severity Stage Prediction
- Predicts disease severity stages
- Uses a dual-head neural network architecture
- Handles limited severity data using synthetic labeling techniques

### Explainable AI Support
- Provides confidence-based outputs
- Generates understandable prediction explanations
- Supports visual interpretation methods

### RAG-Based Recommendation System
- Uses Retrieval-Augmented Generation (RAG)
- Generates textbook-grounded educational responses
- Prevents unsafe medical advice
- Includes stage-based safety warnings

###  Fairness and Accessibility
- Designed to support multiple skin tones
- Mobile-friendly architecture
- Supports future offline deployment capability

## System Architecture

1. User uploads skin image
2. Image preprocessing is performed
3. AI model predicts disease category
4. Severity stage is identified
5. Explainable outputs are generated
6. RAG module retrieves educational information
7. System provides recommendations and safety guidance
